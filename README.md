Nostra – Fashion E-Commerce Website

Nostra is a modern fashion e-commerce front-end website built using HTML, CSS, and JavaScript.
The website showcases clothing collections with a clean UI, product filtering, and search functionality.

This project demonstrates a responsive fashion store layout inspired by modern fashion brands.

Features

• Modern and minimal fashion UI
• Product collection grid
• Category filter system
• Live search functionality
• Responsive layout
• Smooth scrolling navigation
• Product categories such as Hoodies, Jackets, Shirts, Pants, Footwear, Perfume, T-Shirts, and Shorts

Product Categories

The collection page contains products from multiple categories:

• Hoodies
• Jackets
• Shirts
• Pants
• Footwear
• Perfume
• T-Shirts
• Shorts

Each category includes multiple items displayed in a responsive product grid.
